		Version: 0.3 BETA


	Use:
		syntax : function

		ccmd -h	: help

		ccmd -i : install ccmd script (for ccmd to function)

		ccmd -l : print list of custom commands

		ccmd -a <custom cmd> <cmd> : set an alias

		ccmd -r <custom cmd> : remove an alias

		ccmd -L : License info
